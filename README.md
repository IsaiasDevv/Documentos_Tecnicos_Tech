## 🧠 Sobre o Projeto

Este repositório reúne **documentos técnicos acadêmicos** desenvolvidos ao longo do curso de **Análise e Desenvolvimento de Sistemas**.

📌 Todo o conteúdo foi organizado de forma estruturada, separando cada disciplina em diretórios específicos, garantindo:

- ✔️ Clareza  
- ✔️ Organização  
- ✔️ Facilidade de navegação  
- ✔️ Evolução contínua  

💡 O projeto está em constante crescimento, pois o curso ainda não foi finalizado.
